# Stage 01 — Avatar and Positioning

Define the dream client, the gap between where they are and where they want to be, and the competitive landscape they're navigating.

## Inputs

| Source | File/Location | Section/Scope | Why |
|---|---|---|---|
| User input | Conversation | `video-meta.md` from CONTEXT.md questions | Product, format, channel, language, production |
| Hormozi reference | `references/avatar-framework.md` | Full file | How to identify the dream client |
| Hormozi reference | `references/pain-dream-gap.md` | Full file | How to map current state vs. dream outcome |
| Hormozi reference | `references/competitive-landscape.md` | Full file | How to find the blue ocean |
| Product source | (External: landing page, README, docs) | Whatever describes the product | Source material |

## Process

1. Read `video-meta.md` and the three references in this stage.
2. Investigate the product: read the landing page, docs, or whatever the user shared. Note features, current positioning, current audience.
3. Run the avatar exercise from `avatar-framework.md`. Be specific. "Small business owners" is NOT an avatar. "Fitness gym owners with 50-200 members losing $5k/month to no-shows" is.
4. Run the pain-dream-gap exercise from `pain-dream-gap.md`. Surface what they SAY they want vs. what they really want underneath.
5. Run the competitive landscape exercise from `competitive-landscape.md`. Identify direct competitors, indirect, and the status-quo / DIY option.
6. Synthesize into a positioning brief.
7. Run the audit.
8. Save.

## Checkpoints

| After Step | Agent Presents | Human Decides |
|---|---|---|
| 5 | The avatar + 3 candidate positioning angles | Which angle is the strongest before writing the brief |

## Audit

| Check | Pass Condition |
|---|---|
| Specificity | The avatar is described in concrete terms (size, situation, emotion), not generic categories |
| Pain has emotion | The pain section names what they FEEL, not just what's broken |
| Dream is specific | The dream outcome is measurable and visceral, not "be successful" |
| Competitor is named | At least one direct, indirect, and status-quo competitor identified |
